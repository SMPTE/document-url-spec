This document specifies URIs for documents developed by the SMPTE Standards Community. The URIs are intended to:

  • persistently identify documents; and
  • retrieve documents over the Internet.

As illustrated in , URIs are defined for specific versions of documents as well as related collections of documents.

The URIs can also used to retrieve the resource itself: an HTTPS GET request to the URI will return the one or more documents identified by the URI. For example, a GET request to https://pub.smpte.org/doc/ returns an HTML document that contains the list of all available document.

General

URI

URI syntax is specified using the grammar specified at .

Trailing slashes are significant in URIs.

The URI https://pub.smpte.org/doc/st428-12/latest/ is not equivalent to the URI https://pub.smpte.org/doc/st428-12/latest.

The path component of a URI is case-sensitive.

The URI https://pub.smpte.org/doc/st428-12/latest/ is not equivalent to the URI https://pub.smpte.org/doc/st428-12/LATEST/.

Unless specified otherwise, an HTTPS GET request to a URI shall offer to return an HTML document (text/html) and may offer return other media types.

Unless specified otherwise, if a URI resolves to a specific version of a document and the main element of the version is only available as a PDF document, then the HTML document embeds the PDF document.

Publication prefix

The following prefix string is used by URI definitions:

prefix = "https://pub.smpte.org"

Most recent version

All documents

The all-latest URI resolves to a resource that lists the most recent version of all documents.

all-latest = prefix %s"/latest/"

https://pub.smpte.org/latest/

All parts of a document

The latest-all-parts URI resolves to a resource that lists the most recent version of all parts of the document with type type and root number root-number.

latest-all-parts = prefix %s"/latest/" type root-number "/"

https://pub.smpte.org/latest/st429/

Specific part of a document

The latest-part URI resolves to the most recent version of the standard document whose short name is document-short-name.

latest-part = prefix %s"/latest/" document-short-name "/"

https://pub.smpte.org/latest/st429-2/

If the document has no parts, latest-part is equal to latest-all-parts and thus resolve to the same resource.

Most recent publication version

All documents

The all-pub URI resolves to a resource that lists the most recent publication version of all documents.

all-pub = prefix %s"/pub/"

https://pub.smpte.org/pub/

All parts of a document

The all-parts-pub URI resolves to a resource that lists the most recent publication version of all parts of the document with type type and root number root-number.

all-parts-pub = prefix %s"/pub/" type root-number "/"

https://pub.smpte.org/pub/st429/

Specific part of a document

The part-pub URI resolves to the most recent publication version of the standard document whose short name is document-short-name.

part-pub = prefix %s"/pub/" document-short-name "/"

https://pub.smpte.org/pub/st429-2/

If the document has no parts, part-pub is equal to all-parts-pub and thus resolve to the same resource.

All versions

The all-versions URI resolves to a resource that lists all versions of the standard document whose short name is document-short-name.

all-versions = prefix %s"/version/" document-short-name "/"

https://pub.smpte.org/version/st429-2/

Specific version

The specific-version URI resolves to the specific version version-short-name of the document whose short name is document-short-name.

specific-version = prefix %s"/version/" document-short-name "/" version-short-name "/"

https://pub.smpte.org/version/st429-2/20230716-cd/

Package of a specific version

The specific-version-package URI resolves to a single archive file that contains all files that comprise the specific version version-short-name of the document whose short name is document-short-name.

specific-version-package = prefix %s"/package/" document-short-name "/" version-short-name "/"

https://pub.smpte.org/package/st429-2/20230716-cd/

An HTTPS GET request to the specific-version-package URI shall offer to return an ZIP archive document (application/zip) and may offer return other media types.
