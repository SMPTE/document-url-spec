This document specifies URIs for documents developed by the SMPTE Standards Community. The URIs are intended to:

  • persistently identify documents; and
  • retrieve documents over the Internet.

As illustrated in , URIs are defined for specific versions of documents as well as related collections of documents.

The URIs can also used to retrieve the resource itself: an HTTPS GET request to the URI will return the one or more documents identified by the URI. For example, a GET request to https://pub.smpte.org/doc/ returns an HTML document that contains the list of all available standards document.

General

URI

URI syntax is specified using the grammar specified at .

Trailing slashes are significant in URIs.

The URI https://pub.smpte.org/doc/st428-12/latest/ is not equivalent to the URI https://pub.smpte.org/doc/st428-12/latest.

The path component of a URI is case-sensitive.

The URI https://pub.smpte.org/doc/st428-12/latest/ is not equivalent to the URI https://pub.smpte.org/doc/st428-12/LATEST/.

Publication prefix

The following prefix string is used by URI definitions:

prefix = "https://pub.smpte.org/" %s"doc"

All standards document

The all-docs URI resolves to a resource that lists all standards documents.

all-docs = prefix "/"

The resource shall be available as an HTML document (text/html) and may be available in other media types.

All versions of a standards document

The all-versions URI resolves to a resource that lists all versions of the standard document whose short name is document-short-name.

all-versions = prefix "/" document-short-name "/"

The resource shall be available as an HTML document (text/html) and may be available in other media types.

Specific version of a standards document

General

Resources for a specific version of a standards document shall be available as an HTML document (text/html) and may be available in other media types. If the main element of the version is only available as a PDF document, then the HTML document embeds the PDF document.

Latest publication version

The latest-pub URI resolves to the most recent Publication version of the standard document whose short name is document-short-name.

latest-pub = prefix "/" document-short-name %s"/pub/"

Latest version

The latest-version URI resolves to the most recent version of the standard document whose short name is document-short-name, whatever the stage of the version.

latest-version = prefix "/" document-short-name %s"/latest/"

Named version

The specific-version URI resolves to the specific version version-short-name of the standard document whose short name is document-short-name.

specific-version = prefix "/" document-short-name "/" version-short-name "/"
